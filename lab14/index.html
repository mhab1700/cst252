Lab 14 - Loops


Challenge

Our main challenge Was to create a loop that would loop from 1 to 200. With this loop we had to create a function for the numbers 3, 5, and 7 which should print out either Fuzz! Buzz! or Boom! or even a combination based on the numbers being true multiples of either 3, 5, or 7. If there is a remainder when dividing one of these numbers the respective number would get no attribute.


Problems

Results
